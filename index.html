Projects Portfolio

Rare Event Simulation in Finance

This is an academic project whose objective is to put into practice advanced rare event simulation techniques within the context of financial market modeling. The project focuses on applying and testing existing methods—such as adaptive multilevel splitting (AMS) and importance sampling to estimate the probabilities of rare but meaningful events in limit order books (LOBs). These events include market regime transitions (e.g., from bullish to bearish), sudden episodes of illiquidity, and the statistical distribution of the second bid or ask queue when the first one is depleted.

Neural Network Memory Modeling

This research studied how neural networks maintain memory after stimulus removal, using biophysical models (Hodgkin-Huxley, Integrate-and-Fire) with synaptic plasticity. My work on this project involved providing experimental data, creating interpretable features, and refining models through parameter optimization (MSE, DL, and gradient boosting) to align simulations with observed biological behaviors.
